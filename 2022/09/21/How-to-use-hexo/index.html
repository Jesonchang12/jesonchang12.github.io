How to use hexo

##作者：常潭
天气晴朗，万里无云，有点风，有点凉
很困，中午没睡，喝了两杯咖啡
第一次写个人博客，用hexo
挂在GitHub上，有坑，但不多
其中一个是设置了custom domain没成功，然后由于浏览器缓存，
一直会往没设置成功的域名上跳转，ctrl+H清理了浏览器缓存才解决
其他的坑就是，有个缺点，域名只能跟用户名一样，否则域名会很长
因为你改了仓库名之后，域名会在原来的用户名后加上新改的仓库名
还有就是hexo只能用ssh，不能用https，type只支持git
github上是可以指定使用哪个分支作为站点的，这个算一个注意点
hexo重新init之后需要重新配置git组件，npm install –save hexo-deployer-git，
否则会报ERROR Deployer not found: git
暂时就这么多吧，剩下的问题在以后写更多博客的时候应该会遇到，到时候再解决
哦对了，hexo的命令虽然在cmd下也能用，但是只有在git bash里面才能直执行成功，
原因未知
